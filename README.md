# Программы для проверки и обслуживания НЖМД

Набор программ предназначен для проверки, настройки и обслуживания накопителей на жёстких магнитных дисках (НЖМД), подключённых через контроллер по схеме ПК "Вектор-06ц". Запускаются в любой версии МикроДОС, РДС, CP/M, а также в Мониторе-Отладчике на ПК "Вектор-06ц" и "ПК-6128ц++". 

## TESTHDD6
TESTHDD предназначен для проверки работы контроллера НЖМД.

Сделан с использованием алгоритмов тестирования из программ TESTHDD и TESTHDD2 В.Фролова.
Полностью переписаны процедуры обращения к НЖМД, улучшено распознавание характеристик жёских дисков, расширен вывод тестовой информации.

## INITHDD6
Программа для записи конфигурации на НЖМД -- количества секторов/головок/цилиндров и количества дискет. Перед записью конфигурации рекомендую прогнать TESTHDD6 для того, чтобы убедится в исправности контроллера и совместимости подключённого жесткого диска.

Из-за ограничений ПО Вектора значения параметров не могут превышать:
- Секторов: 255 (FFh)
- Головок: 16 (10h)
- Цилиндров: 65534 (FFFEh)
- Секторов в режиме LBA: 16777215 (FFFFFFh)
- Дискет: 10686 (29BEh)

Максимальная доступная на Векторе область диска, соответственно, получается 8 Гб, чего более чем достаточно. На дисках большого объёма рекомендуется уменьшить число дискет и цилиндров до объёма 32-64 Мб, а остальное место использовать для [раздела FAT16](https://zx-pk.ru/threads/36322-hdd-vektora-i-fat16.html).
